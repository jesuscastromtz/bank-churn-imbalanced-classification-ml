# bank-churn-imbalanced-classification-ml
Implementé un sistema avanzado de clasificación para predecir clientes bancarios en riesgo de abandonar, enfocado en resolver el desafío de datos desbalanceados (1:10 ratio). Optimicé F1-score mediante técnicas específicas para clases minoritarias, identificando patrones críticos que permitirían retener clientes de alto valor.
